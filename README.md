## Usage

### Setup

```sh
# clone this repositry with new project name
$ git clone https://github.com/pirosikick/hiden-no-tare.web.git your-project
$ cd your-project

# git init this repositry as new git repositry
$ rm -rf .git README.md
$ git init

# install dependencies
$ npm install
```
